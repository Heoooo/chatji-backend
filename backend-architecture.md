# Chatji Backend Architecture & Technical Highlights

## 🎯 프로젝트 개요
Chatji는 상품 가격 비교 및 핫딜 탐색 서비스로, 외부 쇼핑 API 데이터를 기반으로 사용자에게 최적의 가격 정보를 제공하는 시스템이다.

---

## 🧠 핵심 기술 요소

### 1. 캐시 전략 (Redis)
- 검색 결과 캐싱 적용
- Key: search:{keyword}:{sort}
- TTL 기반 데이터 관리
- 캐시 히트율 개선으로 응답 속도 향상

---

### 2. 멱등성 보장
- 동일 검색 요청 시 동일 결과 반환
- 중복 데이터 저장 방지
- 요청 단위 idempotent 설계

---

### 3. 외부 API 안정성 처리
- timeout 설정
- retry 전략 적용
- fallback 처리 설계

---

### 4. 계층형 아키텍처
- Controller / Service / Client 분리
- 관심사 분리 (Separation of Concerns)
- 유지보수성과 확장성 고려

---

### 5. 데이터베이스 설계
- Product Entity 설계
- 인덱스 기반 조회 최적화
- 중복 데이터 방지 구조

---

### 6. 정렬 및 데이터 처리
- 가격 기반 정렬
- 향후 랭킹 알고리즘 확장 가능

---

### 7. 배치 및 확장 가능 구조
- 가격 히스토리 저장 구조 고려
- 가격 변동 감지 시스템 확장 가능

---

### 8. 비동기 구조 확장 (Future Work)
- 이벤트 기반 알림 시스템
- Producer-Consumer 구조 도입 가능

---

## 🚀 확장 방향

- 가격 변동 감지 시스템
- 사용자 맞춤 알림 기능
- 상품 유사도 분석
- 검색 최적화 (Elasticsearch)

---
